Block Scheduler

Getting Started

Use the tabs on the left to navigate the program.

This program will build you a custom calendar for the year, tailored to your class schedule! You can also generate weekly lesson plan templates under "See it."

Use the build tab to input your class schedule.

Use the customize tab to configure more advanced settings.

The resulting calendar planner will appear under the "See it" tab.

If there's any interruptions like snow days or emergency cancellations, you can edit the schools calendar under "Snow Days."

Printing

The planners are printable!

The program will do its best to make sure each month fits on its own page, however it might not always work things out perfectly.

Be sure to use print preview. If your browser has a shrink to fit option, enable it, and make sure to also enable print backgrounds if you want to see any shading or colors.

Some browsers let you disable the headers and footers, such as the time of printing. It is also recommended that you reduce the margins as much as possible.

If shrink to fit fails to fit each month onto its own page, try adjusting the scale until it does fit. 90% seems to work well.

Each tab is printable, so be sure to select the "See it" tab if you want to print your yearly planner. This program could also be used to print your schedule ("Build it") or a one page calendar ("Snow days").

For best results, use Firefox or Chrome on PC/Windows and Safari on Mac.

Class Schedule

To begin, enter all your classes (preps) into the class list by hitting the "+" button on the left. You can rename a class by doubleclicking it on the left.

After adding a class, select it from the list. Then click in each appropriate cell on the schedule to place it. Use the Eraser to clear cells.

Use the panel above to color code and stylize your schedule, if you like.

Finally, if your schedule looks correct, you should proceed to the "See it" tab!

Only periods with classes in them will appear on the planner under "See it" by default. For example, if the Lunch period has no classes in its row, then nothing will appear for that period in the generated planner.

Double click on a period or day to rename it. Use the tiny buttons on the far right and bottom of the table to add and remove days/periods.

Planner Settings



By default, if a period has no sections/classes, that period will not appear on the planner. "Keep Empty Periods" changes that.

Displaying period numbers may take up space, so disabling them is one way to fit more on each line in the planner.

Save Control


You can delete a schedule or calendar here. You can undo a delete with the restore buttons below until the page is reloaded.

Click export calendar to generate the save file for the current schedule. Only server admins need to do this.

School Calendar

This page is for adding snow days. It can also be used to edit and play with the calendar. Changes will be automatically saved. If you do not want this, disable saving under settings.

If the school calendar is not updated by an administrator, this page can also be used to build the entire calendar yourself.

To edit the calendar, simply select a day type below and click a date on the calendar to set it to that day type.

If you make a mistake and want to forget your changes, you can hit "erase calendar" under the customize tab.

ADVANCED: You can export this calendar to a file from the settings page.

Select a day type below.

The eraser type will remove any calendar settings for a specific date. Select another type, like "holiday" for more info on what that type does.

Holidays are days where school is not in session. The day cycle skips that date.

Set days are days when the day cycle must be set to a certain day. This might happen after an unplanned series of snow days or in the event of cycle makeup days.

Half days are days when school is in session but only for certain periods.

No class days are when school is in session but you don't have your classes. The day cycle keeps counting. This tends to happen during exam weeks or when one campus/building is in session.
